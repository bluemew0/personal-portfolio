Suscord IdleModerator

Become the best Suscord moderator to get hired by Suscord! Suscord IdleModerator is an idle game about moderating servers and gaining clout to progress your moderating career. This game was made using vanilla JavaScript, HTML, and CSS.

I worked on this game as a 2D artist and some of my contributions include:

  • Creating UI mockups
  • Illustrating 10 profile pictures for the game
  • Designing the logo
  • Developing the title invite screen
  • Creating a trailer for the game
  • Using Clip Studio Paint and Figma to create mockups and assets
  • Collaborating remotely with a team of 7 people
