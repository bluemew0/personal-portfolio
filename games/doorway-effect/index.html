      Doorway Effect

      Help Lethe collect their items and escape "The Doorway Effect" to get to school!

      This short, engaging memory game is a simple but fun experience that anyone can enjoy and you can learn to connect with our main character, Lethe. (The doorway effect happens when you pass through different “doorways,” or environments, and your memory declines. More information can be found here.)

      Made by Andy Eng, Ashley Lu, Siwen Tao, and Erick Tung.

      I worked on this browser game as lead artist, art director, and side programmer. You can see my concept art and finished products below. Some of my contributions include:

      • Working with 3 other classmates to brainstorm ideas and develop the game
      • Compiling reference images into a style moodboard
      • Concepting color palettes for the environment and character
      • Concepting character designs
      • Illustrating the title artwork, environment tile set
      • Animating the playable character
      • Implementing animations and adding visual tweak using Phaser 3 JavaScript framework
      • Using Aseprite to animate and Clip Studio Paint to concept and illustrate
      • Developing and styling a website to hold our game on
