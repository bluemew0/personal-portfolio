Digging Doug

Digging Doug is a personal project to learn the process of making a game using Godot. It is a recreation of the "Voltorb Flip" game from Pokemon Soul Silver and Heart Gold into a cute pixel game. I had a lot of fun playing that game and I wish it were an actual mobile app, which is why Digging Doug exists. So far, the basic puzzle mechanics work but a lot of the smaller details need to be refined.

The game itself has a similar base to Minesweeper where there are spots on a grid that either have points or a bomb. However, the information you are given is different and the goal of the puzzle is to find all the 2- and 3- point locations. The game will give you the point total and the total number of bombs in each row and column. Based on that, you must try your best to solve the puzzle.
