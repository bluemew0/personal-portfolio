One Button Games

A series of games made with the same prompt of "One Button" and the Crisp Game Library. With a limited time of 2 weeks to learn this new library, I had to work with 3 people, 1 person, and myself to make 3 games. Listed below are the games made in order and the photos are also in order.

  • Firefly Catch

    Created with Leon Chen, Dylan Higashionna, Rachel Trieu. I worked on creating the jar sprite and implementing it. Because this was the first game using the Crisp Game Library, I focused on just learning my way with the library.

    You can play the game here.

  • Color Beats

    Created with Neo Zhang. Together, we worked on the design of the game and I worked on the visuals of the game. In making this game, we were able to figure out how animations worked and so we added that to this game.

    you can play the game here.

  • Playing with Fire

    For this game, I worked alone, meaning I also had to code the game. I faced some challenges, especially in trying to calculate the circular spawning and the state changes but I think I was able to combine everything I learned about this library into this game.

    You can play the game here.
